# FLUX LoRA Results & Training Analysis

This document provides a comprehensive visual and technical analysis of all generated results for the **FLUX.1-dev (12B)** model. It details the training performance, seen prompt validation (10 prompts before/after), and unseen generalization evaluation (50 prompts before/after).

> All result folders live under: `Results_before_after_training/flux/`

---

## 1. Hyperparameters & Setup

| Parameter | Value |
| :--- | :--- |
| **Base Model** | `black-forest-labs/FLUX.1-dev` |
| **Text Encoders** | T5-XXL + CLIP ViT-L |
| **Hardware** | Multi-GPU (`CUDA_VISIBLE_DEVICES=1,2`), distributed via `accelerate` |
| **LoRA Rank / Alpha** | 32 / 32 |
| **Target Modules** | Cross-attention & Feed-Forward projections (double-stream blocks) |
| **Learning Rate** | `1e-4` |
| **Precision** | `bfloat16` |
| **dataset_repeat** | 20 (→ 200 steps per epoch) |
| **Dataset Size** | 177 training images |
| **Epochs Completed** | Epoch 0 (200 steps) → Epoch 1 (400 cumulative steps) |

### 2.2 Seen Prompts — Key Findings (from interpretation.md)

- **High-Fidelity T5 Parsing:** T5-XXL accurately resolves complex Italian terminology (*Nodo quadro, trama tripla, Corteccia di Giunco*) into correct visual outputs — impossible with CLIP-only encoders.
- **Micro-Texture Realism:** Juniper bark grain and rattan texture are crisp; the VAE reconstructs sharp edges without blurring seen in smaller models.
- **Edge Stitching:** Stitched leather borders are straight and thread-level patterns are well-resolved.
- **Failure — Dimensionality Collapse:** Crossings appear as 2D texture maps rather than true 3D interlocking strands; parallax depth is missing.
- **Failure — Concept Bleeding:** In two-color prompts (e.g., Prompt 5), cross-attention maps have not fully decoupled — color smudging at strand boundaries.
- **Failure — Structural Over-fitting:** All outputs replicate the training dataset's center-focused macro composition, limiting camera-angle variance.

### 3.2 Unseen Prompts — Key Findings (from interpretation.md)

- **Generalization Collapse at Epoch 0:** The model at 200 steps behaves nearly identically to the base FLUX model on unseen prompts. Style transfer is weak.
- **Border / Stitch Failure:** The signature `intrecciami-style` stitched leather borders are largely absent — the model has not yet associated the trigger word with border geometry on unseen prompts.
- **Prior Domination:** For domain-specific terms (*Rinfilo, Uncinetto, Paglia di Vienna, Fettuccia*) the model ignores material definitions and falls back to generic wicker/straw priors.
- **No Micro-occlusion Shadows:** Contact shadows at weft/post crossings are absent — rendering is physically implausible.
- **Structural Geometry Preserved:** The FLUX base model's strong geometric prior keeps grids straight and symmetrical even without LoRA influence.

## 4. Key Takeaways & Recommendations

| Dimension | Seen (Epoch 1) | Unseen (Epoch 0) |
| :--- | :--- | :--- |
| **Border Injection** | ✅ Excellent | ❌ Absent |
| **Material Realism** | ✅ Leather pores, bark grain | ❌ Generic |
| **Grid Geometry** | ✅ Straight & aligned | ✅ Preserved (base prior) |
| **Color Fidelity** | ✅ Accurate | ✅ Partial |
| **3D Depth / Parallax** | ⚠️ Slightly flat | ❌ Flat |
| **Concept Generalization** | ✅ Strong | ❌ Weak — priors dominate |

**Recommendations:**
1. **Extend training to Epoch 2+** (≥ 400 steps on unseen domain) to force cross-attention map updates for border geometry on unseen prompts.
2. **Increase LoRA rank to 64** to capture 3D parallax depth and decoupled material profiles for fettuccia vs. mignon vs. strobel.
3. **Add seen/unseen validation after each epoch** to track generalization vs. memorization trade-off across training budget.
